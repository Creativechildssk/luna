# LUNA v2.0.0

Release date: 2026-04-10

## Summary
LUNA v2.0.0 establishes the Version 2 baseline for the platform. This release combines the refreshed dashboard experience, stronger mobile-first observation tooling, installable PWA support, live satellite identification scaffolding, and a formalized documentation and versioning workflow.

## Highlights
- Redesigned dashboard cards and hierarchy for a cleaner, more product-ready UI.
- Improved moon phase presentation and viewing-score visualization.
- Added Live Identify as a quick-access tool for real-time satellite matching.
- Expanded AR support with fullscreen and streamlined observation flow.
- Improved PWA install prompts and offline shell support.
- Hardened backend dependency and Docker build workflow for cloud deployments.
- Added professional project documentation, wiki content, and semantic version tooling.

## Frontend changes
- Refined dashboard surfaces, stat cards, and timer presentation.
- Improved Moon phase card readability and proportion.
- Added circular viewing-score gauge.
- Improved quick-tool presentation and tab navigation.
- Updated visible UI branding to Version 2.

## Backend and platform changes
- FastAPI app version aligned to `2.0.0`.
- Docker and dependency workflow improved for AWS-style mirrored package installs.
- Semantic versioning policy documented for API-safe releases.
- Added repository script for coordinated version updates across backend, frontend, docs, and UI labels.

## Documentation and process changes
- Rewrote root README as a professional project landing page.
- Added architecture, getting started, API, versioning, and release documentation.
- Added GitHub-ready wiki pages under `wiki/`.
- Added contributor guidance for code, docs, and release work.

## Compatibility notes
- This release marks the v2 baseline.
- Future breaking API changes should increment the major version.
- Backward-compatible feature additions should increment minor.
- Backward-compatible fixes should increment fix.
